CSS can be added to HTML documents in 3 ways:

  • inline-by using the style attribute inside HTML elementsl
  • External-by using a style element in the head section

In CSS, selectors are used to target a specific element or range of elements on a web pages. once an element has been targeted, a style or set of styles can be applied to the element

There is wide range of selectors available.two of the most commonly used are class and ID.both are used to target elements to which a style should be applied.
